# VC Intelligence Dashboard

Next.js frontend for VC intelligence data visualization.

## Features

- **Modern UI** - Next.js 15 with React 19
- **Real-time Search** - Filter by type, state, focus areas
- **Statistics Dashboard** - Visual analytics
- **Responsive Design** - Mobile-friendly

## Quick Start

```bash
npm install
npm run dev
```

## Environment Variables

```bash
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
```
